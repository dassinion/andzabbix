# AndZabbix
Android zabbix client

Support all Zabbix versions:
- 1.x
- 2.x
- 3.x

Features:
* Multiple zabbix servers
* Alarms dashboard
* Charts
* Acknowledgment for alarms
* Configuration of zabbix server
